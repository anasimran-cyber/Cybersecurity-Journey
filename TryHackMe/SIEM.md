# Overview

A Security Information and Event Management (SIEM) platform collects, normalizes, and analyzes security logs from multiple sources to help organizations detect threats, investigate incidents, and improve security visibility.

---

# Learning Objectives

- Understand the role of SIEM.
- Learn how log collection works.
- Explore event correlation.
- Understand threat detection and alerting.

---

# Key Concepts

## What is SIEM?

SIEM combines Security Information Management (SIM) and Security Event Management (SEM) into a centralized platform for monitoring security events.

### Common Log Sources

- Firewalls
- Servers
- Endpoints
- Web Servers
- Authentication Logs
- Network Devices

### Core Functions

- Log Collection
- Log Normalization
- Event Correlation
- Alert Generation
- Threat Detection
- Incident Investigation

### Popular SIEM Solutions

- Splunk
- Microsoft Sentinel
- IBM QRadar
- Elastic SIEM
- Wazuh
