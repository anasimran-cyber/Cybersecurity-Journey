# Overview

Web applications are among the most common targets for cyber attacks. Understanding how web technologies function and where vulnerabilities exist is essential for identifying and preventing security risks.

---

# Learning Objectives

- Understand the basics of web applications.
- Learn the HTTP request and response model.
- Explore common web vulnerabilities.
- Understand secure web development practices.

---

# Key Concepts

## Web Applications

Web applications allow users to interact with services through a web browser.

## HTTP and HTTPS

- HTTP transfers web data.
- HTTPS encrypts communication using TLS.

## Common Web Vulnerabilities

- SQL Injection
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Broken Authentication
- Directory Traversal

## Security Measures

- Input validation
- Output encoding
- Secure authentication
- HTTPS
- Least privilege

# Key Takeaways

- Most modern attacks target web applications.
- Secure coding practices reduce vulnerabilities.
