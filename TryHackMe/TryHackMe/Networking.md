# Overview

Networking is one of the fundamental concepts in cybersecurity. Understanding how devices communicate enables security professionals to identify vulnerabilities, analyze network traffic, and troubleshoot connectivity issues.

This room introduces the basics of computer networking, common network devices, protocols, IP addressing, and how data travels across a network.

# Learning Objectives

- Learn the purpose of networking.
- Identify common networking devices.
- Understand IP addresses and ports.
- Learn about common network protocols.
- Understand how data travels between devices.

# Key Concepts

## What is Networking?

A network is a collection of interconnected devices that communicate and share resources such as files, printers, and internet access.

### Common Types of Networks

- LAN (Local Area Network)
- WAN (Wide Area Network)
- MAN (Metropolitan Area Network)
- WLAN (Wireless Local Area Network)

---

## Network Devices

- Router
- Switch
- Hub
- Firewall
- Access Point
- Modem

Each device has a specific role in forwarding, filtering, or managing network traffic.

---

## IP Address

An IP address uniquely identifies a device on a network.

Examples:

```
192.168.1.10
10.0.0.5
172.16.1.20
```

Two common versions:

- IPv4
- IPv6

---

## Ports

Ports allow multiple network services to run on a single device.

Common ports include:

| Port | Service |
|------:|---------|
| 20/21 | FTP |
| 22 | SSH |
| 23 | Telnet |
| 25 | SMTP |
| 53 | DNS |
| 80 | HTTP |
| 110 | POP3 |
| 143 | IMAP |
| 443 | HTTPS |

---

## Common Protocols

- HTTP
- HTTPS
- FTP
- SSH
- DNS
- DHCP
- SMTP
- TCP
- UDP

Each protocol serves a different purpose in network communication.

---

## TCP vs UDP

- TCP is reliable and connection-oriented.
- UDP is faster but does not guarantee packet delivery.
