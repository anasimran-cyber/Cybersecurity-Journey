# Overview

An Intrusion Detection System (IDS) monitors network or host activity to identify suspicious behavior, policy violations, and potential cyber attacks. Unlike a firewall, an IDS does not block traffic but generates alerts for security teams to investigate.

---

# Learning Objectives

- Understand the purpose of an IDS.
- Differentiate between Host-based and Network-based IDS.
- Learn how IDS detects malicious activity.
- Explore common detection methods.

---

# Key Concepts

## What is an IDS?

An IDS continuously monitors systems or network traffic for indicators of compromise and alerts administrators when suspicious activity is detected.

### Types of IDS

- Network Intrusion Detection System (NIDS)
- Host Intrusion Detection System (HIDS)

### Detection Methods

- Signature-Based Detection
- Anomaly-Based Detection

### Common IDS Solutions

- Snort
- Suricata
- Zeek (formerly Bro)
- Wazuh

# Key Takeaways

- IDS monitors and alerts but does not actively block attacks.
- Signature detection identifies known threats.
- Anomaly detection helps discover previously unknown attacks.
